# common-lib
Common library with shared DTOs, utils, etc.

Для публикации нужен токен доступа github (можно сделать личный или на организацию).

Cтавим в переменные окружения нужные переменные, они подтянутся при публикации:
```shell
export GITHUB_USER=
export GITHUB_TOKEN=
```
а дальше публикация:

```shell
./gradlew publish
```
